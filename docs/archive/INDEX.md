## 💡 常见问题（FAQ）

### Q1: 为什么选择 Zig 而不是 Rust？
**A**: 
- 编译速度更快（开发效率）
- 内存管理更直观（学习曲线）
- 与 C 互操作更简单（FFI 友好）
- 性能相当（实测差距 < 5%）

详见: [RESEARCH.md](RESEARCH.md) 第 1 节

---

### Q2: MCP 会不会被淘汰？
**A**: 
- Anthropic 官方标准（背书强）
- 已有 Claude、Cursor 支持（生态好）
- 类似 LSP（语言服务器协议）的地位（长期价值）

详见: [RESEARCH.md](RESEARCH.md) 第 2 节

---

### Q3: 安全性如何保证？
**A**: 
1. 强制 Transaction Simulation（100% 覆盖）
2. Program 白名单验证（只允许可信协议）
3. 金额限制（单笔/每日上限）
4. Session Key（临时授权）
5. 代码审计 + Bug Bounty（外部验证）

详见: [ARCHITECTURE.md](ARCHITECTURE.md) 第 3 节

---
