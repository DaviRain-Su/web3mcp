✨ 核心功能
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. discover_contracts   - 发现可用合约 (动态)
2. discover_chains      - 列出支持的链
3. call_contract        - 调用任何合约 (动态)
4. get_balance          - 跨链余额查询
5. transfer             - 跨链转账
6. get_transaction      - 交易查询
7. get_block            - 区块查询
8. sign_and_send        - 签名并发送


🚀 使用方法
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. 配置 Claude Desktop:
   ~/Library/Application Support/Claude/claude_desktop_config.json
   {
     "mcpServers": {
       "omniweb3": {
         "command": "<项目路径>/scripts/run.sh"
       }
     }
   }

2. 重启 Claude Desktop

3. 开始使用:
   - "有哪些智能合约可用？"
   - "查询我的 BSC 测试网余额"
   - "在 BSC 上调用 PancakeSwap 合约"
