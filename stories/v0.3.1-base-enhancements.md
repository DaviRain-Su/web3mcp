# Story: v0.3.1 Base Enhancements (Solana + EVM)

> 补齐 Solana 与 EVM 的基础只读能力：账户、交易、区块、Token 余额等。

## Goals
- 补齐 Solana 基础 RPC 能力（账户/交易/Token）
- 补齐 EVM 基础 RPC 能力（交易/区块/Nonce/Gas/Token）
- 提供只读工具优先（更安全、可扩展）
