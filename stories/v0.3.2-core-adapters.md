# Story: v0.3.2 Core Adapter Refactor

> 将工具层抽象到 core adapters + wallet，提升可扩展性。

## Goals
- core/chain 成为统一入口
- core/wallet 提供统一的 key & sign 接口
- tools 不直接调用 RPC/SDK
